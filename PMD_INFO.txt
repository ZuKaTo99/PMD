PMD – Project Manager Dashboard

Aktueller Stand: V0.5

Beschreibung:
PMD ist eine .NET MAUI Blazor Hybrid Desktop-App zur Verwaltung von Softwareprojekten.

Aktueller Schwerpunkt:
Projektprüfung / Scanner-Grundstein

Stand V0.5:
- App startet sauber
- Build funktioniert ohne Fehler
- Startziel ist Windows Machine
- Projektprüfung-Seite funktioniert
- Ordnerauswahl über FolderPicker funktioniert
- manuelles Pfadfeld bleibt zusätzlich erhalten
- Scanner liest Projektordner rekursiv ein
- typische Projekt-/Buildordner werden übersprungen
- Prüfergebnis zeigt Projektname, Projektordner, Prüfzeitpunkt, Dateianzahl und Gesamtgröße
- geprüfte Ordner werden gezählt
- Prüfdauer wird angezeigt
- übersprungene Ordner werden angezeigt
- Hinweise/Warnungen werden gesammelt und angezeigt
- Dateitypen werden zusammengefasst angezeigt
- größte Dateien werden angezeigt
- gefundene Dateien werden sortiert angezeigt
- Dateiliste ist aktuell auf 50 Einträge begrenzt

Bewusst noch nicht eingebaut:
- SQLite
- Projektstände speichern
- Vergleich alter/neuer Projektstände
- Änderungsbericht
- Kanban
- Rust-Core
- KI-Funktionen

Nächster geplanter Bereich:
Projektstände vorbereiten.

Ein Projektstand soll später ein gespeichertes Prüfergebnis eines Projektordners sein.
Damit kann PMD später zwei Projektstände vergleichen.
